
##  ShoppyGlobe - E-commerce React App

Welcome to **ShoppyGlobe**, a basic yet powerful e-commerce application built using **React**, **Redux**, and **manual CSS styling**. This project demonstrates product listing, cart management, routing, state management, and responsive design â€” all in one sleek package.

---

##  Features

*  Product Listing with Search
*  Product Detail View
*  Cart Functionality with Add/Remove
*  Redux for State Management
*  React Router for Navigation
*  Lazy Loading for Performance
*  Responsive Manual CSS (No Tailwind)
*  Custom 404 Not Found Page
*  Error Handling for Fetch Failures
